Course Welcome

Welcome to Miami Dade College School of Engineering and Technology's Secure Academy Cyber Security course. It is a pleasure to have you onboard this challenging, informative and self-paced class. There are twelve learning modules you will be required to complete along with competency based assessments. Each module will provide you a step by step approach to mastering cyber security techniques using the Gaia Operating System.


Subject Matter Expert

Kim Winfield

Mr. Kim Winfield is a technical instructor at Check Point Software Technologies. Check Point Software Technologies is the largest network cyber security organization. Over the course of 17 years, Mr. Winfield has designed and developed multiple cyber security programs and courses. Winfield is a graduate of Evangel University where he earned his Master's degree.

   Course Description and Competencies

Course Description

This Cybersecurity Fundamentals Online Course will provide learners with principles of technology that frame and define cybersecurity. Learners will gain insight to the importance of cybersecurity and an application of concepts. This competency-based self-guided format will provide a dynamic learning experience where users can explore foundational cybersecurity principles, security architecture, risk management, attacks, incidents, and IT and IS technologies.

Course Competencies

Upon the completion of this course you will be able to comprehend and apply knowledge to:

  1. The Check Point Gaia Operating System
  2. Security Rulebase Management
  3. Security Policy Management
  4. Threat Prevention
  5. Threat Emulation/Threat Extraction
  6. ClusterXL
  7. CoreXL and SecureXL
  8. Identity Awareness
  9. Application Control/URL Filtering
  10. Mobile Access
  11. IPSec VPN
  12. System Monitoring
