   Module Introduction

How do you administer a Mobile Access solution for SSL VPN connectivity from remote clients through the Check Point Mobile Access Blade? How can you use SecureWorkspace to secure remote user desktops and create access to internal web applications using the Mobile Access Blade?


  Mobile Access allows you to quickly select remote users access to internal web or mail applications, through a web browser, mobile device, or remote access client.

Mobile Access

In this module you will learn how to administer the Mobile Access for SSL VPN connectivity from remote clients through the Check Point Mobile Access Blade. Finally, you will use SecureWorkspace to secure remote user desktops and creating access to internal web applications using the Mobile Access Blade.


Learning Objectives

Upon the completion of this module you will be able to:

  1. Administer the Mobile Access solution for SSL VPN connectivity from remote clients through the Check Point Mobile Access Blade
  2. Use SecureWorkspace to secure remote user desktops and creating access to internal web applications using the Mobile Access Blade
