   Module Introduction

How do you install the Check Point GAiA™ operating system? How do you navigate the Check Point Gaia™ OS file structure? How do you perform a backup of the Check Point GAiA™ OS?


  Check Point GAiA™ is a unified secure platform for all Check Point appliances, open servers and virtual gateways that helps consolidate security infrastructure and simplify management  

Check Point Software Technologies

Please complete the   

Check Point GAiA™ OS: installation, navigation and backup

In this module you will learn how to install the Check Point GAiA™ operating system, using both command line installation and the Web User Interface. You will also master how to navigate the file structure of the Check Point GAiA™ operating system. Finally, you will acquire the skills to perform a backup of the Check Point GAiA™ operating system.


Learning Objectives

Upon the completion of this module you will be able to:

  1. Install the Check Point GAiA™ operating system, using both command line installation and the WebUI
  2. Navigate the file structure of the Check Point GAiA™ operating system
  3. Perform a backup of the Check Point GAiA™ operating system
